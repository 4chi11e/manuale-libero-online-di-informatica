## Il linguaggio Python
Python è un linguaggio di programmazione multi-paradigma ad uso generale, sviluppato originariamente da Guido van Rossum alla fine degli anni '80 ed rilasciato per la prima volta nel 1991. Supporta la programmazione orientata agli oggetti, imperativa, funzionale e i modelli procedurali. Ha un sistema di tipi dinamico e una gestione automatica della memoria (tramite garbage collection).

La filosofia di progettazione di Python enfatizza la leggibilità del codice, in particolare attraverso l'uso di rientri significativi (indentazione) per delimitare i blocchi di codice. La sua vasta libreria standard viene spesso descritta come "batterie incluse", fornendo strumenti adatti a moltissimi compiti, dallo sviluppo web alla scienza dei dati, dall'automazione alla scrittura di semplici script.

Negli ultimi anni Python ha visto una crescita straordinaria ed è oggi il linguaggio di riferimento assoluto per ambiti all'avanguardia quali l'Intelligenza Artificiale, il Machine Learning, la data science e il calcolo scientifico, oltre a rimanere uno strumento popolarissimo per lo sviluppo web back-end.
